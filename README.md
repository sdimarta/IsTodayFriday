# Is It Friday? 🤔

> A simple, no-nonsense website that tells you if today is Friday – based on **your local time**.

This is my first experimental project using **spec-kit**, **opencode**, and the **Qwen3.6 Plus Free** model.  
It answers the most important question of the week: *"Is it Friday yet?"*

## 🚀 Features

- ✅ Detects user's **local date and time**
- ✅ Displays a clear **YES / NO** answer
- ✅ Responsive, minimal design
- ✅ Pure client-side JavaScript – no backend required

## 🧠 How it works

1. The page loads and reads the user's system clock using `new Date()` in JavaScript.
2. It checks the `.getDay()` method (where 5 = Friday).
3. Displays:
   - A big happy message if it's Friday 🎉
   - A simple "not yet" message if it's not Friday
4. All logic runs locally – no data is sent anywhere.

## 🛠️ Built with

- HTML5, CSS3, vanilla JavaScript
- [spec-kit](https://github.com/your-link-to-spec-kit) – for specification-driven development
- [opencode](https://github.com/your-link-to-opencode) – as the AI coding assistant
